### 2. 性能优化 (4项)

#### HTTP 客户端优化
```go
// 优化前
Transport: &http.Transport{
    DisableKeepAlives: true,  // 禁用连接复用
}

// 优化后  
Transport: &http.Transport{
    MaxIdleConns:        100,              // 全局最大空闲连接
    MaxIdleConnsPerHost: 10,               // 每个主机最大空闲连接
    IdleConnTimeout:     90 * time.Second, // 空闲连接超时
    DisableKeepAlives:   false,            // 启用 Keep-Alive
}
```

**预期收益**:
- 减少 TCP 连接建立/关闭开销
- 降低延迟 20-50%
- 提高并发处理能力
