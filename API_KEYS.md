## FOFA

**官网**: https://fofa.info/

### 获取步骤

1. **注册账号**
   - 访问 https://fofa.info/
   - 点击右上角"注册"按钮
   - 填写邮箱和密码完成注册
   - 验证邮箱

2. **登录系统**
   - 使用注册的邮箱和密码登录

3. **获取 API Key**
   - 登录后，点击右上角用户名，进入"个人中心"
   - 在左侧菜单中选择"API 管理"或"API密钥管理"
   - 页面会显示您的 API Key 和 Email
   - 记录以下信息：
     - Email: 注册时使用的邮箱
     - API Key: 类似 `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` 的字符串

### 配置示例

在 UniMap Light 中配置：
```
FOFA: [粘贴您的 API Key]
FOFA Email: [输入您的注册邮箱]
```

### 注意事项

- ⚠️ **免费版限制**: 免费账号每月有查询次数限制
- ⚠️ **API Key 安全**: 不要泄露给他人，可以在个人中心重置
- 💡 **VIP 会员**: 付费会员可以获得更多查询配额和高级功能

### 常见问题

**Q: API Key 在哪里查看？**
A: 个人中心 → API 管理

**Q: 免费版有限制吗？**
A: 有，每月查询次数有限制，具体查看个人中心配额显示

## Hunter (鹰图)

**官网**: https://hunter.qianxin.com/

### 获取步骤

1. **注册账号**
   - 访问 https://hunter.qianxin.com/
   - 点击"注册"按钮
   - 支持手机号或邮箱注册
   - 完成注册验证

2. **登录系统**
   - 使用注册的账号登录

3. **获取 API Key**
   - 登录后，点击右上角头像进入"个人中心"
   - 在左侧菜单中找到"API管理"或"用户信息"
   - 查看或创建 API Key
   - 记录 API Key: 类似 `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 配置示例

在 UniMap Light 中配置：
```
Hunter: [粘贴您的 API Key]
```

### 注意事项

- ⚠️ **实名认证**: 部分功能可能需要实名认证
- ⚠️ **查询限制**: 免费版有每日查询次数限制
- 💡 **积分系统**: Hunter 使用积分系统，查询消耗积分

### 常见问题

**Q: 如何获得更多积分？**
A: 可以通过签到、完成任务或充值获得

**Q: API Key 有效期是多久？**
A: 通常长期有效，除非手动重置

## ZoomEye

**官网**: https://www.zoomeye.org/

### 获取步骤

1. **注册账号**
   - 访问 https://www.zoomeye.org/
   - 点击"Sign Up"或"注册"
   - 使用邮箱注册
   - 验证邮箱

2. **登录系统**
   - 使用注册的邮箱和密码登录

3. **获取 API Key**
   - 登录后，点击右上角用户名
   - 进入"个人中心"或"Profile"
   - 找到"API Key"或"密钥管理"选项
   - 查看或生成 API Key
   - 记录 API Key: 类似 `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`

### 配置示例

在 UniMap Light 中配置：
```
ZoomEye: [粘贴您的 API Key]
```

### 注意事项

- ⚠️ **配额限制**: 免费用户有每月查询次数限制
- ⚠️ **API Key 格式**: ZoomEye 的 API Key 通常是 UUID 格式
- 💡 **VIP 权益**: 付费用户可获得更多配额和高级数据

### 常见问题

**Q: 如何查看剩余配额？**
A: 在个人中心可以查看当前配额使用情况

**Q: API Key 丢失怎么办？**
A: 可以在个人中心重新生成新的 API Key

## 通用建议

### 安全最佳实践

1. **保护 API Key**: 
   - 不要在公开场合分享
   - 不要提交到 Git 仓库
   - 定期更换

2. **测试 API Key**:
   - 获取后先在官网测试是否可用
   - 检查配额是否充足

3. **多引擎配置**:
   - 建议至少配置 2 个引擎
   - 可以互相补充，提高覆盖率

### 免费用户建议

如果您使用免费账号：

1. **合理规划查询**:
   - 避免无效查询浪费配额
   - 先用小范围测试，再扩大

2. **选择合适引擎**:
   - 不同引擎数据侧重点不同
   - 根据需求选择最合适的

3. **利用去重功能**:
   - UniMap Light 会自动去重
   - 多引擎查询不会浪费额外配额

### 企业用户建议

如果您是企业用户：

1. **考虑付费套餐**:
   - 更高的查询配额
   - 更完整的数据
   - 更好的技术支持

2. **多账号管理**:
   - 可以注册多个账号分散使用
   - 注意遵守各平台使用条款

3. **数据合规**:
   - 了解各平台的数据使用协议
   - 确保合规使用

## 故障排查

### API Key 无法使用

1. 检查是否正确复制（无空格、换行）
2. 检查账号是否已激活
3. 检查配额是否已用完
4. 尝试重新生成 API Key

### 查询返回错误

1. 检查网络连接
2. 检查 API Key 是否过期
3. 查看引擎官网是否正常
4. 联系引擎客服支持
